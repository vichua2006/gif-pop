# GIF Pop

Collect and save your favorite GIFs to never lose them again! A lightweight Electron app for managing your personal GIF collection. Add your own GIFs, jpg, png as local files, organize with custom names for ease of search, and quickly search & paste them anywhere using a global keyboard shortcut.

Intended usage in mind is to be able to use WeChat emojis in other messaging apps, but feel free to use it for anything you want.

**NOTE: only Windows is supported for now**

## Features

- Add your own GIFs, jpg, png as local files
- Organize with custom names for ease of search
- Global hotkey to quickly search and paste
- Fully offline - all data stored locally

## Development

```sh
npm install
npm run dev
```

## Electron Build

```sh
npm run electron:dev
```
